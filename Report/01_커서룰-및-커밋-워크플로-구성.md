## 요약

SHealth BMI 실습 프로젝트에 Cursor 프로젝트 규칙(`.cursor/rules/`), 커밋 전 Report·Prompting 절차, `/commit` 슬래시 명령을 추가했다.

## 수행 내용

### 1. 프로젝트 규칙 (`.cursor/rules/`)

| 파일 | 적용 | 내용 |
|------|------|------|
| `shealth-project.mdc` | always | README 도메인(BMI·연령대·보정)·실습 목표 |
| `java-refactoring.mdc` | `src/main/java/**` | 코드 스멜 제거·SRP·네이밍 |
| `java-tests.mdc` | `src/test/**` | JUnit 5 TC 항목 |
| `commit-workflow.mdc` | intelligent | 커밋 시 Report→Prompting→Git |
| `response-style.mdc` | always | 한국어 응답·최소 diff |

### 2. 커밋 워크플로

- 이전 `.cursorrules`의 커밋 절차를 `commit-workflow.mdc`로 이전
- `AGENTS.md`에 규칙·명령 인덱스 추가

### 3. 슬래시 명령

- `.cursor/commands/commit.md` → 채팅 `/commit` 실행 시 동일 1→2→3 절차

### 4. `/commit` 테스트 (본 Report)

- `Report/01_…`, `Prompting/01_…` 생성 후 Git 커밋으로 워크플로 검증
